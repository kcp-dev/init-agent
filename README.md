The kcp Init Agent is a Kubernetes controller for initializing newly created workspaces by creating
any number of Kubernetes objects in it.

## Documentation

Please visit [https://docs.kcp.io/init-agent](https://docs.kcp.io/init-agent) for the latest
documentation.
